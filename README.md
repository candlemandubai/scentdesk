# ScentDesk

**The fragrance industry's open intelligence terminal.**

Real-time market data, regulatory updates, raw material tracking, new launches, and career opportunities — all in one place. Think Bloomberg Terminal, but for fragrance professionals.

**[scentdesk.app](https://scentdesk.app)**

---

## Hey, you're reading the source code

If you're here, you're probably a developer, a fragrance nerd, or both. Either way, welcome.

We built ScentDesk because we love this industry and wanted a single dashboard to stay on top of everything — market moves, regulatory changes, new launches, ingredient trends, and career opportunities. The fragrance world is a $125B+ ecosystem with so much happening every day, and we thought it deserved a dedicated intelligence tool.

This project is open source because we believe in building together. The fragrance community has always been generous with knowledge, and ScentDesk is our way of contributing back. The more people using and improving it, the better it gets for everyone.

### What ScentDesk tracks

- Market intelligence & M&A news
- New fragrance launches & industry developments
- Raw materials & supply chain updates
- Home fragrance market (candles, diffusers, room sprays)
- IFRA / EU / FDA regulatory updates
- Industry events & trade shows
- Fragrance houses & ingredient suppliers
- Perfumery schools & education
- Jobs & career opportunities
- Perfumer spotlights & podcasts

### Built with

- [Next.js](https://nextjs.org) — React framework
- [Tailwind CSS](https://tailwindcss.com) — Styling
- [Zustand](https://zustand-demo.pmnd.rs/) — State management
- [Lucide](https://lucide.dev) — Icons
- [Recharts](https://recharts.org) — Charts
- Deployed on [Vercel](https://vercel.com)

### Running locally

```bash
git clone https://github.com/candlemandubai/scentdesk.git
cd scentdesk
npm install
npm run dev
```

Open [localhost:3000](http://localhost:3000) and you're in.
